ETWS Testing Overview

Testing ETWS involves verifying that messages are correctly broadcast and received by mobile devices in the affected area. It also involves ensuring that the message content, such as the alert level and repetition period, is correctly formatted and functional.

Testing Process

  • Verify message generation at CBC.
  • Test message delivery via eNodeB.
  • Ensure mobile devices receive alerts within specified time.
